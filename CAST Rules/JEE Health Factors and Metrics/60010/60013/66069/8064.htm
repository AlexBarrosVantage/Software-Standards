Assignment operator should assign all non-static members of the target object

Technology

C Language, C++

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

8

Rationale

Not assigning all data members in 'operator=' leads to the loss of information and can make the target object in an incoherent state.

Description

This rule searches for C++ assignment operators that do not assign all the non static members of the class.

Remediation

Assign non-static members to the target object.

Output

Associated to each C++ assignment operator with violations, the Quality Rule provides: - The number of missing static members - Bookmarks for violation patterns found in source code: - assignment operator - missing static member

Total

Number of C++ assignment operators
