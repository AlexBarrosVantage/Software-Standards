Rationale

For readability aspects, naming conventions must be respected

Description

Names of Exceptions should end with X, start with an uppercase character and should not include any underscore. The suffix value is a parameter that can be changed at will

Remediation

Change name according to the naming convention

Output

This report lists all Exceptions with names not ending with X, with first letter not in uppercase or having an underscore. It provides the following information: Exception full name

Total

Total number of classes that represent an Exception.
