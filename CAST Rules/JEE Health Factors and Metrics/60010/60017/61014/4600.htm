Avoid using 'java.System.exit()'

Technology

JEE

Unified Status

N/A

XXL Status

Critical Quality Rule

Weight in Parent Tech. Criterion

2

Rationale

Using 'java.System.exit()' is prohibited since it goes against the Exception error handling model.

Description

Java artifacts should not use 'java.System.exit()' for application deployed on a Web Server or an Application Server (when web.xml and/or ejb-jar.xml file are present)
