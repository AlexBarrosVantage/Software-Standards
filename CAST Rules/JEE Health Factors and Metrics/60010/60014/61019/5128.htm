Technology

Cobol

Rationale

Having complex queries in Cobol Programs can cause performance problems.

Description

Queries with sub queries is considered complex.

Remediation

Try to reduce the complexity of queries or the number of complex queries.

Output

This report lists all Cobol Programs with with subqueries. It provides the following information: Cobol Program full name, number of subqueries
