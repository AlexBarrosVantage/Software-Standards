Rationale

Having complex queries in Functions can cause performance problems.

Description

Avoid Functions / Methods with queries on more than 4 Tables. Queries with more than 4 Tables is considered complex. Changing the threshold value requires Metric Assistant configuration update.

Remediation

Review the query to reduce the number of involved Tables.

Output

This report lists all Functions / Methods with queries on too many Tables. It provides the following information: Function / Method full name, number of Tables
