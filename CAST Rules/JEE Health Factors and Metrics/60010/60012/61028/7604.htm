Rationale

Maintainability of the code is facilitated if there is documentation in the code. This rule will ensure there are comments within the Artifact.

Description

ABAP classes should be documented. This reports shows all the ABAP classes with no comments.

Remediation

Add comments into class defintion to explain what is their goal and how they work.

Output

This report lists all ABAP classes that are not documented. It provides the following information: ABAP Class full name
