Objects in Universe using inexistent Table/Column

Unified Status

N/A

Critical Quality Rule

Yes

Weight in Parent Tech. Criterion

9

Rationale

It is possible that some objects use a table or a table column that does not exist anymore. This will not produce an error when saving the universe but will generate an error at run time when a report will use this object.

Description

List all objects in universes where the table/column does not exist anymore.

Remediation

Delete the object (or recreate the missing table/column).

Sample
Object of type dimension based on column MyTable.MyColumn where MyColumn does not exist anymore.
Remediation sample
Delete the object from the universe.
Ensure that no report use this object. Update those reports if needed.
Output

List all objects in universes where the table/column does not exist anymore.

Total

Number of objects in all universes
