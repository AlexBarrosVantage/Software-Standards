Rationale

The higher the number of reference to an Artifact, the more difficult the maintenance as all referencing Artifacts will have to be updated or tested.

Description

Avoid Artifacts with High Fan-In (Fan-In > X). The Fan-In of an Artifact is the number of other Artifacts that are referencing it. When computing the Fan-In of an Artifact, multiple accesses to it from the same Artifact are counted as one access. The threshold level is a parameter that can be changed at will.

Remediation

Reduce the number of references to the Artifact

Output

This report lists all Artifacts with High Fan-In. It provides the following information: Artifact full name, Fan-In value
