Avoid Procedure Paragraphs that contains no statements

Technology

Rationale

For maintainability aspect, avoid procedure paragraphs that contains no statements

Description

This rule searches for Cobol programs containing empty paragraphs.

Remediation

Check if the paragraph is referenced. If yes, then see why this empty paragraph is called. Otherwise remove it.

Output

Associated to each COBOL Program with violations, the Quality Rule provides: - The number of violation patterns - Bookmarks for violation patterns found in source code: - Empty paragraph
