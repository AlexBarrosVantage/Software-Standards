Weight in Parent Tech. Criterion

Rationale

Presence of synonyms that are based on non existent object will generate error at execution time when accessed. Even if synonyms is authorized at creation time, the execution time will generate error

Description

This metric identifies synonyms that are defined on objects which no longer exist. This is syntactically possible. Indeed through a series of CREATE synonym statements, a synonym could be defined and in fact refer to itself

Remediation

Apply clean up in your code

Reference

Oracle documentation

Output

This report lists all synonyms that are defined on objects which no longer exist. It provides the following information: The list of each synonym that match the criterion. The name of each synonym that is part of the full name.
