Rationale

Large methods are more difficult to understand, and are a sign of a bad modularity of the code.

Description

Methods should not have more than X lines of code. The threshold is a parameter and can be changed at will.

Remediation

Review the design of the method to reduce its size

Output

This report lists all Methods with more than X lines of code It provides the following information: Method full name, number of lines of code
