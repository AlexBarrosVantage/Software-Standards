Avoid Classes with a High Public Data Ratio (C# .Net)

Rationale

Public fields are accessible by any other Class, therefore their values can be modified at any time, without control by the Class itself. This goes against OO encapsulation concepts.

Description

Avoid Classes with a High Public Data Ratio greater than X %. Public Data Ratio is the percentage of public fields among all fields. Properties are not considered as fields. The threshold is a parameter and can be changed at will.

Remediation

It is necessary to change the field visibility to private and provide appropriate accessors.

Output

This report lists all Classes with a Public Data Ratio greater than X %. Const data members should not be considered (as public). It provides the following information: - Class full name, PDC
