Declare as static all Methods not using Instance Fields

Technology

.Net, C#

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

5

Rationale

When an object is created: * Memory is allocated to all the fields * All super class fields are also allocated memory * All sub class fields, super class fields are initialized . * The constructor is invoked. Using a static avoid to create an object that takes resources when it is unnecessary.

Description

All methods that don't use instance fields and/or method will be reported except if they have been extended in a subclass or inherit from their parents. For all .Net Classes, including .Net Forms : If a method doesn't use an instance field or const, it should be declared static except if the method is overriden or override another method.

Remediation

Make this method static.

Sample
class Example {
    protected double pi = 3.14;

    int min (int x, int y) { // VIOLATION
      return (x>y?y:x);
    }

    double getPi() {  // OK
       return pi;
    }
Remediation sample
class Example {
    protected double pi = 3.14;

    static int min (int x, int y) { // FIXED
      return (x>y?y:x);
    }

    double getPi() {  // OK
       return pi;
    }
Output

All methods that don't use instance fields (either const or not) and/or method will be reported except if they have been extended in a subclass or inherit from their parents.
