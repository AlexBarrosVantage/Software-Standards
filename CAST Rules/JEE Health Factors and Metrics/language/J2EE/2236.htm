Avoid use of standard SQL API

Technology

JEE

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

8

Rationale

Standard SQL APIs java.sql.* and javax.sql.* are symptoms of direct access to tables.

Description

Reports all pages that reference a class, interface or method whose full qualified name begin with 'java.sql.' or 'javax.sql'

Remediation

Review the architecture design to invoke the layer in charge of database access.

Sample
<%@ page import = "java.sql.*" %>
<% Connection myConnection =null; //VIOLATION
      Statement myStatement =null; //VIOLATION

      ResultSet myResultSet =null; //VIOLATION

      javax.sql.DataSource myDataSource = null; //VIOLATION
%>

// 4 violations will be reported
Output

This report lists all JSP pages using standard SQL APIs It provides the following information: - Page name - the number of violation

Total

the number of JSP pages
