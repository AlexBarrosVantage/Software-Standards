Rationale

To ease changes to the presentation layer, JSP Pages should always be accessed through their tiles definition. The Tiles framework makes it very easy to add and remove view components and to rearrange things.

Description

Reports any direct references to JSP pages except those from Tiles.

Output

This report lists all JSP pages that are not accessed through their tiles. It provides the following information: - Page name - JSP Tiles Definition or JSP Tiles Attributes

Total

The total is the total of JSP pages. When the tiles technology is not used (no object of type "Tiles Configuration File" is present) the ADG must not display anything
