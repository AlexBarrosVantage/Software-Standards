Rationale

For maintainability and readability reasons, artifacts should not have too many parameters.

Description

Avoid artifacts with more than X parameters. The threshold X is a parameter and it can be changed at will.

Remediation

Review the design of the artifacts to reduce number of parameters.

Output

This report lists all artifacts with more than X parameters. It provides the following information: Artifact full name, number of parameters

Total

number of artifacts that have at least 1 parameter
