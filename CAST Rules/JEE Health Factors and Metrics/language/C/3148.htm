An include file must contain at most one Class definition

Technology

C++

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

5

Rationale

Having several class definitions in a header file makes it more complex to understand/navigate.

Description

This rule searches for include files (i.e. a file that is included at least by another file) that contain multiple class definitions.

Remediation

If several classes are defined in a header file, split class definitions into multiple header files.

Sample
FILE Header.h
class One {
   int amember1;
};

class Two {
   int amember2;
}
Remediation sample
FILE One.h
class One {
   int amember1;
};

FILE Two.h
class Two {
   int amember2;
}
