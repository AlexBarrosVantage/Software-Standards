Rationale

Having complex queries in Artifacts can cause performance problems.

Description

Avoid SQL Artifacts with queries on more than 4 Tables. Queries with more than 4 Tables are considered complex. Changing the threshold value requires Metric Assistant configuration update.

Remediation

Review the query to reduce the number of involved Tables.

Output

This report lists all Artifacts, except Views, with queries on too many Tables. It provides the following information: Artifact full name
