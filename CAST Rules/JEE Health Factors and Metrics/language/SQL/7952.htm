Rationale

Presence of a synonym with several definitions induces code complexity.

Description

This metric retrieves synonyms that have two definitions: one private and one public

Remediation

Reconsider the use of synonym with multiple definitions: One idea could be to rename one of them

Reference

Oracle documentation

Output

This report lists all the synonyms with two definitions It provides the following information: The ordered list of all synonyms that have both private & PUBLIC definition. The name of each synonym that is part of the full name.

Total

The total procedure counts the number of synonyms that are part of the analyzed source code
