Rationale

Maintainability of the code is facilitated if there is documentation in the code. This rule will ensure there are comments within the Artifact.

Description

ABAP user-exits should be documented. This reports shows all the ABAP user-exits with no comments.

Remediation

Add comments into user-exit to explain what is their goal and how they work.

Output

This report lists all ABAP user-exits that are not documented. It provides the following information: ABAP UserExit full name
