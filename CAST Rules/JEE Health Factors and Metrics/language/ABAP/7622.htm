Rationale

Unreferenced members of classes like any other example of dead code make maintenance more difficult and more expensive.

Description

This metric retrieves all members of ABAP class that are not referenced i.e. not used by ABAP code.

Remediation

Check if the members of class are going to be used. If not, then remove them from the class.

Output

This report lists all members of ABAP classes that are not referenced. It provides the following information: ABAP Member full name
