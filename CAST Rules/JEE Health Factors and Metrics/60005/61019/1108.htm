Avoid Functions with queries on too many Tables (C Lang.)

Technology

C Language

Unified Status

Not Unified

Critical Quality Rule

No

Weight in Parent Tech. Criterion

9

Rationale

Having complex queries in Functions can cause performance problems.

Description

Avoid Functions with queries on more than 4 Tables. Queries with more than 4 Tables is considered complex. Changing the threshold value requires Metric Assistant configuration update.

Remediation

Review the query to reduce the number of involved Tables.

Output

This report lists all Functions with queries on too many Tables. It provides the following information: Function full name, number of Tables
