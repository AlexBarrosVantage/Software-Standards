Avoid unreferenced Includes

Technology

ABAP

Unified Status

Critical Quality Rule

Weight in Parent Tech. Criterion

4

Rationale

Unreferenced includes like any other example of dead code makes maintenance more difficult and more expensive (modification of unused code).

Description

This metric retrieves all ABAP includes that are not referenced i.e. not included either in a program or another include.

Remediation

Check if the Artifact is truly unnecessary and remove Artifacts that are not used.

Output

No additional value associated to each ABAP Include with violation.

Total

Number of ABAP Includes
