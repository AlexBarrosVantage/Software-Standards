Avoid Programs/Includes with too many Forms

Technology

ABAP

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

4

Rationale

High number of Forms decreases the understanding of the program logic. Thus, maintenance will become difficult.

Description

Programs / includes with too many forms are difficult to maintain. This report shows all the ABAP programs and Includes defining more than X forms. X is a parameter and can be changed at will

Remediation

Check if some Forms are similar and try to factorize them.

Output

This report shows all Abap Programs and Abap Includes having too many Forms. It provides the following information: Abap Program/Includes fullname, number of Forms

Total

Number of Abap Programs and Abap Includes
