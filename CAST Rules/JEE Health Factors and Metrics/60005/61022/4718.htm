Critical Quality Rule

Weight in Parent Tech. Criterion

4

Rationale

Packages must have a reasonable size when measured in number of Classes/interfaces.

Description

A Package should not have more than X and less than Y Public Classes/interfaces. The parameters X and Y, maximum number and minimum number can be customized at will.

Output

This report lists all Packages having more than X and less than Y Public Classes or Interfaces. It provides the following information: - Package full name - Number of Classes and Interfaces
