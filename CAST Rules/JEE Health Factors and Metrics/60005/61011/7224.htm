Rationale

High Raw SQL Complexity denotes an artifact that either contains a very large number of queries and/or very complex queries. The artifacts high Raw SQL Complexity can be difficult to change and can potentially cause performance problems.

Description

Avoid Artifacts with High RAW SQL Complexity (SQL Complexity greater than X). RAW SQL Complexity measures the total number of tables used in FROM Clauses (Note: a single table used X times counts X). The threshold is a parameter and can be changed at will. Having complex queries in artifacts can cause performance problems.

Remediation

Review the design of the query to reduce the number of tables used in the FROM clause
