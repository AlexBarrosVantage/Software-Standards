Avoid using Global Variables

Rationale

The key disadvantages of using Global Variables are: Side effects - Hidden access - for example when a local variable has the same name as the global variable. Indiscriminate access - it is impossible to prevent access to the variable, therefore is may be difficult to make changes in the future.

Description

Global Variables should not be used.

Remediation

Use parameters in functions and subs.

Output

This report lists all uses of Global Variables.
