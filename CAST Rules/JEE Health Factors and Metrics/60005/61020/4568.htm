Avoid declaring Public Instance Variables

Technology

JEE

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

Yes

Weight in Parent Tech. Criterion

7

Rationale

A Public Instance Variable can be accessed by any other Class, therefore its value can be modified at any time, without control by the Class itself. This is contrary to OO Encapsulation concepts.

Description

Public Instance Variables should not be used. Instance Variables are Non Static Fields.

Remediation

Change the Field's visibility or make it Static Final.

Output

This report lists all Public and Non Static Fields. It provides the following information: Field full name
