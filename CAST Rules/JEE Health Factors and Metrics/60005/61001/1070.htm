Avoid header files circular references

Rationale

Header files often include other header files. This leads to compilation problems when there are circular references in Header files. For example: Header1.h includes Header2.h which includes Header1.h. This will be detected on "n" levels.

Description

This rule searches for circular references between C header files.

Remediation

Review the header file dependency and remove circular calls.
