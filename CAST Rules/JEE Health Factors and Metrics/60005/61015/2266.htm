Rationale

For readability aspects, naming conventions must be respected.

Description

All files should have standard file extensions. The list of standard, authorized extensions is a parameter that can be changed at will.

Remediation

Change the file extension according to the recommended naming convention.

Output

This report lists all files with a non-standard extension. It provides the following information: File full name
