Rationale

In order to ease change and to hide platform specific information from any potential misuse, it is highly recommended to use SAP ABAP logical paths instead of hardcoded paths.

Description

This metric retrieves all ABAP Artifacts using hardcoded paths instead of using logical file paths

Remediation

Use standard SAP ABAP logical paths instead of hardcoded paths and the standard ABAP function "FILE_GET_NAME" instead.

Reference

Creating and Defining Logical Paths http://help.sap.com/saphelp_46c/helpdata/en/fc/eb3deb358411d1829f0000e829fbfe/content.htm
