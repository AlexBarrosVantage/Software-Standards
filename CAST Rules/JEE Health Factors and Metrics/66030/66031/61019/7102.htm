Avoid Artifacts With Queries on more than 4 Tables (ABAP)

Technology

ABAP

Unified Status

Not Unified

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

9

Rationale

Artifacts containing queries on a large number of tables can cause performance problems.

Description

This rule searches for Artifacts having OpenSQL queries on more than 4 tables.

Remediation

Try to reduce the complexity of queries or the number of complex queries.

Output

This report lists all Artifacts having OpenSQL queries on more than 4 tables. It provides following information: - Artifact full name

Total

Number of ABAP Artifacts
