Rationale

For readability aspects, naming conventions must be respected => Names of Cursors defined as global in a Package should start with <%PARAM%>

Description

List of schema cursors that have name not started with <%PARAM%> The prefix value is a parameter that can be changed at will

Remediation

Change name according to the naming convention

Output

This report lists all Cursor names not starting with the specific prefix. It provides the following information: - Cursor name, -Cursor full name

Total

total is number of cursor defined into the schema
