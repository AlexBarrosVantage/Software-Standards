Rationale

For readability aspects and to respect SAP standard, naming conventions must be respected.

Description

ABAP Functions should begin with Z. This reports shows all the ABAP Functions not starting with Z. The prefix is a parameter and can be changed at will.

Remediation

Change name according to the naming convention.

Output

This report lists all ABAP Functions that does not respect the naming conventions. It provides the following information: ABAP Function full name

Total

Number of ABAP Functions
