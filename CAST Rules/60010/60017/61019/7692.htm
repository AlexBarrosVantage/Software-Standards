Each opened file must be closed

Technology

Cobol

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

7

Rationale

Each file used by a program must be properly opened and closed. It is especially important for temporary datasets that are allocated in the step that executes the Cobol programs. If the file is not closed explicitly, then data can be lost.

Description

This rule searches for Cobol programs in which a file is opened and not closed.

Remediation

For each Cobol programs using files, check if files are closed. If not, then add a CLOSE statement at the end of the program.

Reference

IBM Enterprise Cobol - Performance Tuning

Sample
PROCEDURE DIVISION
       0000-MAIN.
           OPEN OUTPUT DTA-FILE.
           PERFORM 1000-PROC1.
           STOP RUN.
           
       1000-PROC1.
           ...
           MOVE VAR-01 TO DTA-FILE-REC.
           WRITE DTA-FILE-REC.
Remediation sample
PROCEDURE DIVISION
       0000-MAIN.
           OPEN OUTPUT DTA-FILE.
           PERFORM 1000-PROC1.
           CLOSE DTA-FILE.
           STOP RUN.
           
       1000-PROC1.
           ...
           MOVE VAR-01 TO DTA-FILE-REC.
           WRITE DTA-FILE-REC.
