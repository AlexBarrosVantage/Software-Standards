Description

Since EJB 2.0, EJB Session should be accessed through their local remote Interface instead of their remote Interface. This avoids the need for the JNDI lookup, which improves the performance.

Output

This report lists all the Methods using the EJB Session remote Interfaces. It provides the following information: - Method full name - Session Java Bean Name
