Rationale

Instantiating a Boolean object consumes memory and CPU but does not bring value.

Description

Java artifacts should not instantiate Booleans. Example of instantiation: 'Boolean b = new Boolean();'

Remediation

Use Boolean.TRUE or Boolean.FALSE instead.
