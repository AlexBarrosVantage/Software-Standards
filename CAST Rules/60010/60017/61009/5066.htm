Avoid using PERFORM ... THROUGH | THRU

Technology

Cobol

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

6

Rationale

When using PERFORM ... THROUGH, someone may add a new paragraph between existing paragraphs within the bounds of the statement. It results in unexpected behaviors difficult to debug.

Description

This rule searches for Cobol programs using PERFORM ... THROUGH | THRU statements.

Remediation

Create a section containing the paragraphs performed by PERFORM THROUGH and replace the calling statements by PERFORM to section.

Sample
PROCEDURE DIVISION.
A.
    PERFORM PAR-BEG THROUGH PAR-END.
    STOP RUN.

PAR-BEG.
    IF VAR1 = 99
        MOVE 0 TO VAR1
    END-IF

PAR-PROC.
    MOVE 50 TO VAR2.

PAR-END.
    EXIT.
Remediation sample
PROCEDURE DIVISION.
A.
    PERFORM PAR.
    STOP RUN.

PAR SECTION.
PAR-BEG.
    IF VAR1 = 99
        MOVE 0 TO VAR1
    END-IF

PAR-PROC.
    MOVE 50 TO VAR2.

PAR-END.
    EXIT.
