Avoid use of "truncate table"

Rationale

Although "truncate table" statement can be desirable under some circumstances from a performance point of view it is problematic because deletion are not logged. Thus the log backups become useless.

Description

Avoid using "truncate table",

Remediation

Ensure that the truncate table is necessary. Otherwise use delete.

Output

This report lists all uses of "truncate table" in database objects containing code: SP, Functions, and Triggers.

Total

Total number of database objects containing code: SP, Functions, and Triggers.
