Rationale

Built-in remote addresses cause problems when the target is moved. Avoid of hardcoded network resources (e.g., IP addresses, URLs, etc.)

Description

This quality rule reports all artifact that contains hardcoded: - URL - Path - IP adress

CISQ Rule 26: http://it-cisq.org/wp-content/uploads/2012/09/CISQ-Specification-for-Automated-Quality-Characteristic-Measures.pdf
