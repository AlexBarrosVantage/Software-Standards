Avoid disabling source code inspection

Technology

ABAP

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

5

Rationale

Source code inspection checks for bad practices and potential performance issues. For maintenance, performance and robustness reasons, it is better to avoid deactivating SAP source code inspection rules.

Description

This rule checks ABAP source code for searching directives that allow the deactivation of specific checks performed by the Code Inspector tool. These directives are: "SET EXTENDED CHECK OFF" and ""#EC...".

Remediation

Check if these directives are absolutely necessary and, if not, remove them from the source code.

Sample
Example 1:
SET EXTENDED CHECK OFF
<Coding>
SET EXTENDED CHECKS ON

Example 2:
DATA: I TYPE I. "#EC NEEDED
I = 4.
BREAK-POINT.    "#EC NOBREAK
WRITE 'Hugo'.   "#EC NOTEXT
FORM AB.        "#EC CALLED
ENDFORM.
Remediation sample
* SET EXTENDED CHECK OFF
<Coding>
* SET EXTENDED CHECKS ON

Example 2:
DATA: I TYPE I. 
I = 4.
BREAK-POINT.    
WRITE 'Hugo'.   
FORM AB.        "#EC CALLED
ENDFORM.
