Rationale

For maintainability and readability reasons, Interfaces should not have too many Methods.

Description

Interfaces should have less than X Methods. The threshold is a parameter and can be changed at will.

Remediation

Review the Interface design to reduce the number of Methods

Output

This report lists all the Interfaces having more than X Methods. It provides the following information: - Interface full name - Number of Methods
