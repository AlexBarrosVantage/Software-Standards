Avoid direct use of database objects (ASP - JavaScript)

Technology

ASP

Unified Status

Not Unified

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

8

Rationale

ASP/COM model recommends a multi-layer architecture. ASP must communicate with the business layer which in turn will communicate to the data layer through a data access layer.

Description

Pages should not directly use Database Objects

Remediation

Use business layer and data layer instead

Output

This report lists all pages directly using Database Objects. It provides the following information: Page name, Database object name
