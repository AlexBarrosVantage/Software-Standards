Rationale

Action Classes should only be called by Action Mappings to respect struts pattern design

Description

All Action classes that are referenced in the struts-config.xml through the tag 'action-mapping' (in struts 1.x) or in struts.xml and its included files through the tag 'action' (in struts 2.x) that are invoked by other Java code are listed except when the caller is in the inheritance tree of the Action class.

Remediation

Use the action mapping table in struts configuration file instead of direct call.

Output

This report lists all methods that call Action Classes It provides the following information: - Non action mapping class method that call the action class except those in its inheritance tree - Action Class full name

Total

number of methods or objects that call struts actions classes
