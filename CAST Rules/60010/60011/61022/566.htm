Rationale

For maintainability and readability reasons, Classes should not have too many Methods.

Description

This rule searches for C++ classes that contain too many methods. The threshold is a parameter and can be changed at will.

Remediation

Review the Class design to reduce the number of Methods

Associated to each C++ Class with violations, the Quality Rule provides: - The number of methods
