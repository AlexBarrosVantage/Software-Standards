Avoid Programs with too many includes

Technology

ABAP

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

4

Rationale

High number of Includes decreases the visibility in terms of data definitions and processes. Program maintenance will become difficult.

Description

This reports shows all the ABAP programs including more than X includes. X is a parameter and can be changed at will.

Remediation

Check if all includes are necessary.

Sample
REPORT z_repko .

INCLUDE z_inctop.
INCLUDE z_incsel.
INCLUDE z_incfor.
INCLUDE z_inctok.
INCLUDE z_incgo.
INCLUDE z_incdown.

START-OF-SELECTION.

  PERFORM f_initial.
...
Output

Associated to each ABAPProgram with violations, the Quality Rule provides: - The number of includes
