Rationale

When declaring a Variable with 'Dim Var As Object' the compiler will not assign a type. It worsens performance level and prevents compilation to detect typing errors.

Description

Avoid declaring a Variable with 'Dim var As object'

Remediation

Use real type to declare the Variable. E.g.: 'Dim Var_Nam as String = new'

Output

This report lists all variable declarations without variable typing
