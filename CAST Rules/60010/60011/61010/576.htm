Rationale

For maintainability and readability reasons, classes should not have too many constructors.

Description

This rule searches for C++ classes that have too many constructors. The threshold is a parameter that can be changed at will.

Remediation

Review the class design to reduce the number of constructors.

Associated to each C++ class with violations, the Quality Rule provides: - The number of constructors
