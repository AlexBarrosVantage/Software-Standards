Avoid Includes with low comment/code ratio

Weight in Parent Tech. Criterion

Rationale

Maintainability of the code is facilitated if there is documentation in the code. This rule will ensure there are comments within the Artifact.

Description

ABAP includes should be documented. This reports shows all the ABAP includes with less than a defined comment/code ratio. This ratio is a parameter of the rule and can be changed at will.

Remediation

Enrich Artifact code with comments.

Output

Associated to each ABAP Include with violations, the Quality Rule provides: - The comment/code ratio

Total

Number of ABAP Includes
