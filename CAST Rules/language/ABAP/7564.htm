Rationale

For readability aspects, naming conventions must be respected.

Description

SAP database table should begin with Y or Z followed by T (ex: ztorder). This reports shows all the SAP database tables not starting with YT or ZT. The prefix is a parameter and can be changed at will.

Remediation

Change name according to the naming convention.

Reference

Enhancing the Quality of ABAP Development, Meijs-Krouwels-Heulmans-Sommen, SAP Press ISBN 1-59229-030-2

Output

This report lists all ABAP database tables that does not respect the naming conventions. It provides the following information: ABAP Table full name
