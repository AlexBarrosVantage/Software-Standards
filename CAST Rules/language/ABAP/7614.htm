Avoid Methods with a low comment/code ratio (ABAP)

Technology

ABAP

Unified Status

Not Unified

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

1

Rationale

Maintainability of the code is facilitated if there is documentation in the code. This rule will ensure there are comments within the Artifact.

Description

ABAP methods should be documented. This reports shows all the ABAP methods with less than a defined comment / code ratio. This ratio is a parameter of the rule and can be changed at will.

Remediation

Enrich Artifact code with comments.

Output

Associated to each ABAP Class Method/Event Method/Constructor with violations, the Quality Rule provides: - The comment / code ratio

Total

Number of ABAP Methods, ABAP Event Methods and ABAP Constructors
