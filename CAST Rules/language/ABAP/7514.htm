Rationale

For maintainability aspect, avoid ABAP Includes that contains no statements.

Description

This metric retrieves all ABAP Includes that contains no statements. Includes generated by SAP for function groups (ending by "UXX") are excluded.

Remediation

Check if the Include is referenced. If not, then you can envisage to remove it.

Output

This report lists all ABAP includes that contain no statements. It provides the following information: - ABAP Includes full name
