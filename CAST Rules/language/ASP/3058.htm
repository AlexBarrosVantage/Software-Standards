Rationale

By using a style sheet, you ensure that the display is the same across the website. It allows to centralize maintenance of pages look and feel.

Description

This metric detects the pages where a Style sheet (*.css) is not directly or explicitly used. Style sheet (*.css) should be used (instead of no css or hardcoded style definition like style= ).

Remediation

Use a style sheet

Output

This report lists all pages not directly or explicitly using a style sheet. It provides the following information: Page name

Total number of pages
