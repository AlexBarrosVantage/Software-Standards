Rationale

High Raw SQL Complexity denotes a program that either contains a very large number of queries and/or very complex queries. The programs high Raw SQL Complexity can be difficult to change and can potentially cause performance problems.

Description

Avoid Cobol Programs with High RAW SQL Complexity (SQL Complexity greater than X). Having complex queries in programs can cause performance problems. The threshold level is a parameter that can be changed at will.

Remediation

Try to reduce the complexity of queries or the number of complex queries.
