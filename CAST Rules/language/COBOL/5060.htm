Avoid using MERGE

Rationale

COBOL merges are known to be inefficient on zOS environment. It is better to use external merge.

Description

This rule searches for Cobol programs using the MERGE statement.

Remediation

Remove all unnecessary MERGE action and replace them by external tool.

Output

Associated to each COBOL Program with violations, the Quality Rule provides: - The number of violation patterns - Bookmarks for violation patterns found in source code: - MERGE statement
