Avoid undocumented Paragraphs

Technology

Cobol

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

7

Rationale

Maintainability of the code is facilitated if there is documentation in the code. This rule will ensure there are comments within the Artifact.

Description

This rule searches for Cobol programs containing paragraphs without any comment. Note: Empty paragraphs and paragraphs that contain EXIT verb only are excluded from violation. The EXIT statement does nothing and then cannot be considered as a real statement

Remediation

Add comments into paragraphs to explain what is their goal and how they work.

Output

Associated to each COBOL Program with violations, the Quality Rule provides: - The number of violation patterns - Bookmarks for violation patterns found in source code: - Undocumented paragraph
