Avoid High Response for a Class (C# .Net)

Technology

.Net, C#

Unified Status

Not Unified

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

8

Rationale

If a large number of methods can be invoked in response to a message, the testing and debugging of the Class becomes more complicated since it requires a greater level of understanding required on the part of the tester.

Description

Avoid High Response for a Class (RFC > X) RFC is the total number of local methods and remote methods called by methods in the Class. The threshold is a parameter and can be changed at will.

Remediation

Reduce the number of local methods and remote methods called by methods in the Class

Reference

A Metrics Suite for Object Oriented Design Chidamber, S.R. and C.F. Kemerer IEEE Transactions on Software Engineering, 1994

Output

This report lists all Classes with High Response for a Class. It provides the following information: Class full name, RFC
