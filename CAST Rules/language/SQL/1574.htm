Use at most one statement per line

Weight in Parent Tech. Criterion

Rationale

There should not be more than one statement per line of code. Statements are separated by ';' When writing many statements on the same line, code becomes more difficult to read and therefore more difficult to maintain.

Description

List of all Procedures and Functions with more than 1 statement per line

Remediation

Edit code to have one single statement per line

Output

This report lists all Procedures and Functions with more than 1 statement per line It provides the following information: - Function / Procedure full name, - number of lines with more than 1 statement

Total

total is number of SQL procedures and functions that are stored in the schema
