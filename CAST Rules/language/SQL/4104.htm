Rationale

For readability aspects, naming conventions must be respected => Table names should not be longer than X characters.

Description

List of database tables that have more than X characters. The length is a parameter that can be changed at will.

Remediation

Change name according to the naming convention

Output

This report lists all Table names longer than <%PARAM%> characters. It provides the following information: - Table fullname

Total

total is number of tables into the database
