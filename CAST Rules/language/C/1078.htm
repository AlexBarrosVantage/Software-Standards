Avoid undocumented Header files

Technology

C Language, C++

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

7

Rationale

Maintainability of the code is facilitated if there is documentation in the code.

Description

This rule searches for C/C++ header files that do not contain any comments to document the content.

Remediation

Insert comments to document the objective and the content of the header files.

Output

Associated to each C/C++ header file with violations, the Quality Rule provides: - header file name

Total

Number of C/C++ header files

1078
