Rationale

For maintainability and readability reasons, interfaces should not have too many methods.

Description

This rule searches for interfaces that have too many methods. The threshold is a parameter that can be changed at will. Note: C++ interfaces are pure virtual classes.

Remediation

Review the interface design to reduce the number of methods.

Output

Associated to each C++ interface with violations, the Quality Rule provides: - The number of methods
