Proper overriding of 'clone()'

Technology

JEE

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

Yes

Weight in Parent Tech. Criterion

9

Rationale

A call to 'super.clone()' ensures the clone behavior will still work.

Description

When overriding the 'clone()' Method, a call to 'super.clone()' is necessary. Note that where a class 'A' that has a field 'a' of type 'A' and is calling a.clone() without calling super.clone() will not be reported as a violation.

Remediation

Review the Method definition

Output

This report lists all Methods overriding 'clone()' and not calling 'super.clone()'. It provides the following information: Method full name
