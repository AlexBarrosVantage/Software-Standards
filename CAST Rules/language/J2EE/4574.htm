Rationale

Deprecated Methods or Fields may not be available in the next version.

Description

List all Deprecated Methods or Fields. In Java, Deprecated objects are defined with the JavaDoc tag @deprecated or the annotation @deprecated available since JSE 5. (to obtain accurate results, the java archive rt.jar must also have been analyzed). In VC++ it reports objects using the deprecated __declspec declaration or pragma. In GCC it reports objects with the deprecated attribute If the reason/remediation of the deprecation is described in the source code, it is also displayed.

Reference

http://msdn.microsoft.com/en-us/library/044swk7y%28VS.80%29.aspx http://msdn.microsoft.com/en-us/library/c8xdzzhh%28v=VS.80%29.aspx

Output

This report lists all the Methods using a deprecated Method It provides the following information: - Method full name - deprecated Object full name If the reason/remediation of the deprecation is described in the source code, it is also displayed.
