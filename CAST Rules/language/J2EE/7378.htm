Avoid include JavaScript Files

Technology

JEE

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

4

Rationale

JavaScript includes are sent by the server each time the page that include it is loaded. If you use the HTML tag <link> to refer to this file, the browser doesn't have to query for the static content for every page request. Most popular browser like IE, FireFox, Safari, Opera first check their local cache for the static file/resource and only if they don't find do they make a request to the web server for the same.

Description

JavaScript Files must be referenced through the HTML tag SCRIPT

Remediation

For JSP: create a link (html tag) instead of an include. It will be stored in the client cache instead of being sent each time the page is loaded. For other limit the size.

Sample
<%@include file="script.js" %> // VIOLATION
or 
<jsp:include file="script.js"> // VIOLATION
Remediation sample
<SCRIPT TYPE="text/javascript" SRC="script.js'> // FIXED

or

<SCRIPT LANGUAGE="JavaScript" SRC="script.js'> // FIXED
Output

This report lists all JSP pages that include a JavaScript file.

Total

number of JSP files that refer to JavaScript Files
