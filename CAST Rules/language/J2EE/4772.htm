Avoid SQL Artifacts with SQL statement including Subqueries (JEE)

Technology

JEE

Unified Status

Not Unified

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

7

Rationale

Having complex queries in SQL Artifacts can cause performance problems. Queries with Subqueries are considered complex.

Description

SQL Artifacts should not use SQL statement including Subqueries (at least subqueries should be avoided).

Remediation

Review the design of the query to avoid if possible Subqueries

Output

This report lists all SQL Artifacts with SQL statement including Subqueries. It provides the following information: SQL Artifact full name
