Provide accessors to Private Fields

Technology

JEE

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

4

Rationale

To respect OO encapsulation concepts, private fields should always be accessed through accessors.

Description

Each private field should have a setter and a getter except final static fields. Accessors are identified using the following java bean naming conventions: "set", "get" or "is" followed by the name of the field with the first char in uppercase (case sensitive). Methods not using this naming convention are not considered as accessors.

Remediation

Write a getter and setter to each private field

Reference
