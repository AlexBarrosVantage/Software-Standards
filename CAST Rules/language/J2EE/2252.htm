Avoid direct access to database Tables (JSP)

Technology

JEE

Unified Status

Not Unified

XXL Status

N/A

Critical Quality Rule

Yes

Weight in Parent Tech. Criterion

8

Rationale

Direct access to database Table does not respect the multi-layer architecture principles. Furthermore, accessing Tables prevents the control at the database level of accesses. E.g.: use of non-optimized query against the database, use query that don't follow the data logic and that create inconsistency.

Description

Pages should not access directly database Tables

Remediation

Use data layer instead

Output

This report lists all Pages having direct access to database Tables. It provides the following information: Page full name, accessed Table
