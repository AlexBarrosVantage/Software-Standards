Avoid "SELECT *" queries and use of dbms_output.put_line

Please note that the Quality Rule Avoid "SELECT *" queries will be violated even if the select * clause in the PL/SQL source code is located in a dbms_output.put_line procedure. For example:

CREATE OR REPLACE PROCEDURE TEST_PROC
IS
BEGIN
SELECT a FROM tab WHERE b = 'c';
dbms_output.put_line('select * from tab') ;
END;
