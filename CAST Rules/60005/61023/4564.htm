Avoid large Methods - too many Lines of Code (JEE)

Technology

JEE

Unified Status

Not Unified

Critical Quality Rule

No

Weight in Parent Tech. Criterion

6

Rationale

Large Methods are more difficult to understand, and are a sign of a bad modularity of the code.

Description

Methods should not have more than X lines of code (comments are not taken into account). The threshold is a parameter and can be changed at will.

Remediation

Review the design of the Function to reduce its size

Output

This report lists all Methods with more than 100 lines of code It provides the following information: Method full name, number of lines
