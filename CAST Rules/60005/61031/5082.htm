Avoid Procedure Sections that contain no Paragraph

Technology

Cobol

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

4

Rationale

For maintainability aspect, avoid procedure sections that contain no paragraphs. Statements should be located in paragraphs.

Description

This rule searches for Cobol programs containing sections without any paragraph in PROCEDURE division.

Remediation

Create at least one paragraph in the section to include all its statements. Furthermore, it could be then the opportunity to simplify the implementation of the logic and the readability of the code by structuring the statements in several paragraphs.

Output

Associated to each COBOL Program with violations, the Quality Rule provides: - The number of violation patterns - Bookmarks for violation patterns found in source code: - Section without any paragraph
