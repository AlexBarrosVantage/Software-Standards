Rationale

Leaving unused and unnecessary is detrimental to code readability. It is also an issue for modularity and component reuse.

Description

Measures the amount of unused and unnecessary Java imports compared to all Java imports.

Remediation

Remove unused and unnecessary Java imports. This remediation is now automated in the major Java development environments.

Output

This report lists Files having unused imports. It provides the following information: File full name
