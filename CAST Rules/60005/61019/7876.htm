Avoid using SELECT ... FOR UPDATE

Technology

ABAP

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

9

Rationale

Some addition to Open SQL statements automatically bypass the buffer. Bypassing buffer means that SELECT statement will not go to buffered results but always directly to the database. This can impact the performance significantly.

Description

This rule search for SELECT statement using the FOR UPDATE addition.

Remediation

Check if the FOR UPDATE addition is really necessary and use this addition carefully.

Sample
SELECT employee_name 
FROM employees 
WHERE employee_id = 123 
FOR UPDATE.
