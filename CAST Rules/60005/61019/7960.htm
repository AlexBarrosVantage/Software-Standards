Rationale

Presence of a looping chain of synonyms will generate error at execution time when accessed. Even if a looping chain of synonyms is authorized at creation time, the execution time will generate "ORA-01775 looping chain of synonyms" error

Description

This metric retrieves synonyms that are part of a looping chain of synonym. Indeed through a series of CREATE synonym statements, a synonym could be defined and in fact refer to itself

Remediation

Change at least one synonym definition to cut the circular list

Reference

Oracle documentation, for example

Sample
For example, the following definitions are circular:

SQL> create synonym s1 for s2;
Synonym created.
SQL> create synonym s2 for s3;
Synonym created.
SQL> create synonym s3 for s4;
Synonym created.
SQL> create synonym s4 for s1;
Synonym created.
Output

This report lists all the ordered looping chain of synonyms : It provides the following information: the ordered list of each synonym that is part of the looping chain. The name of each synonym that is part of the full name,

Total

The total procedure counts the number of synonyms that is part of the analyzed source code
