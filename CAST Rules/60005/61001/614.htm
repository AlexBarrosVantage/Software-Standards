Avoid direct usage of database tables (C++)

Technology

C++

Unified Status

Not Unified

XXL Status

N/A

Critical Quality Rule

Yes

Weight in Parent Tech. Criterion

8

Rationale

Direct access to database tables prevent the centralization of SQL-Queries at the database level. This can leads to performance problem, the inability to tune the queries and to duplicate code.

Description

Files should not have embedded SQL access to database tables.

Remediation

Data access should be done through a data layer and through stored procedures.

Output

This report lists all files having accesses to database tables. It provides the following information: File full name, accessed table
