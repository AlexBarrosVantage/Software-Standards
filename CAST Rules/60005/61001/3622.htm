Avoid direct access to Database Tables (C# .Net)

Technology

.Net, C#

Unified Status

Not Unified

XXL Status

N/A

Critical Quality Rule

Yes

Weight in Parent Tech. Criterion

8

Rationale

Direct access to database Table prevents the control at the database level of accesses. E.g.: use of non-optimized query against the database.

Description

Applications should not access directly database Tables

Remediation

Use Stored Procedures instead

Output

This report lists all Objects having direct access to database Tables. It provides the following information: Object full name, accessed Table

Total

Total number of .net objects accessing the database
