Rationale

Using the Document.all collection is slow when retrieving its different objects into HTML, HTM, JSP, JS files as it collects - and loads in memory - all items.

Description

Avoid using "Document.all" JScript code.

Remediation

E.g.: Rather use: var temp = document.foo temp.innerText = "Hello" Instead of document.all['foo'].innerText = "Hello"
