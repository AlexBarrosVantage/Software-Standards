Avoid Function pools with more than 20 functions

Technology

ABAP

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

4

Rationale

Each time a function of a function pool is accessed, the entire function pool is put in memory. For performance reason, it is better to avoid having too many functions in the same function pool.

Description

This rule searches for Function Pools having more than 20 functions.

Remediation

Try to have coherent Function Pools.

Output

Associated to each Function Pools with violations, the Quality Rule provides: - The number of Function Modules

Total

Number of Function Pools
