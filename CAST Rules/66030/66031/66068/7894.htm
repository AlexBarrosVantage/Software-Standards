Avoid using LOOP INTO, use LOOP ASSIGNING instead

Technology

ABAP

Unified Status

N/A

XXL Status

N/A

Critical Quality Rule

No

Weight in Parent Tech. Criterion

8

Rationale

LOOP ... ASSIGNING is faster than LOOP ... INTO if field-symbols are modified. In this specific case, the system automatically update data in internal table.

Description

This rule checks all Abap artifacts accessing tables with a LOOP INTO construct.

Remediation

Replace LOOP INTO constructs by LOOP ASSIGNING constructs and use field-symbols.

Sample
LOOP AT itab INTO wa_tab.
  ...
  wa_tab-fld1 = 'X'.
  MODIFY itab FROM wa_tab.

ENDLOOP.
Remediation sample
LOOP AT itab ASSIGNING <fs>.
  ...
<fs>-fld1 = 'X'.

ENDLOOP.
